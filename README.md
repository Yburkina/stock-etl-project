# Stock Data ETL (Beginner Project)

## What it does
A simple ETL pipeline that:
1) Extracts daily stock prices from Yahoo Finance (yfinance)
2) Cleans and reshapes the data
3) Loads the result into CSV and Excel

## Tech stack
Python, pandas, yfinance, openpyxl

## How to run
```bash
pip install -r requirements.txt
python etl_stock_prices.py
